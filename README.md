# Insurance-Data-Insights
Designed and developed interactive dashboards using Power BI, Tableau, and Microsoft Excel, incorporating KPI cards,  revenue funnel analysis, slicers, and comparative Target vs Achieved vs New Business metrics.

The Insurance Analytics Dashboard is an end-to-end data analytics project designed to transform raw insurance data into meaningful, actionable insights. This project integrates SQL, Excel, Power BI, and Tableau to analyze insurance sales performance, revenue trends, opportunity pipelines, and operational KPIs through interactive and visually intuitive dashboards.

The solution helps stakeholders monitor targets vs achievements, evaluate new and renewal business, track sales funnel performance, and assess account executive productivity for informed decision-making.
